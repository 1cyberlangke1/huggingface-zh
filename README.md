# Hugging Face 中文化插件

个人自用 Hugging Face 界面中文化 Tampermonkey 脚本。

> 灵感来源：[maboloshi/github-chinese](https://github.com/maboloshi/github-chinese)

## 功能

- 中文化 Hugging Face 网页界面（导航、按钮、标签、提示等）
- 支持模型详情、数据集详情、 Spaces、设置页面等
- 所有翻译均在本地完成，无外部 API 调用
- 依赖 Tampermonkey / Greasemonkey 运行

## 安装

1. 安装 [Tampermonkey](https://www.tampermonkey.net/) 浏览器扩展
2. [点击安装](huggingface-zh.user.js)
3. 访问 Hugging Face 页面即可

## 说明

- 仅提供**简体中文**翻译
- 翻译由 **AI 辅助完成**，可能存在错误翻译或不准确的情况
- 主要翻译界面 UI 文本，用户生成内容（模型卡片、数据集描述等）不在翻译范围内
